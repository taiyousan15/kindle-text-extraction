## Hash Calculation Details

### Why MD5?
- **Fast**: ~0.01-0.05 seconds per screenshot
- **Sufficient**: We only need duplicate detection, not cryptographic security
- **Deterministic**: Same image always produces same hash
- **Collision Resistant**: Extremely unlikely for different pages to have same hash

### Hash Calculation Code
```python
def _calculate_screenshot_hash(self) -> str:
    screenshot_bytes = self.driver.get_screenshot_as_png()
    return hashlib.md5(screenshot_bytes).hexdigest()
```

### Example Hash Values
```

```

Even a 1-pixel difference results in a completely different hash!
